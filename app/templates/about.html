About This Project

Project Overview

The CVI Risk Predictor is a data science tool designed to support Community Violence Intervention (CVI) organizations in making proactive, data-driven decisions about resource allocation and intervention strategies. By analyzing publicly available crime, socioeconomic, and environmental data at the neighborhood level, this tool provides actionable risk assessments that help CVI teams anticipate and prevent violence.

Purpose & Mission

Community Violence Intervention programs are highly effective at reducing violence through trusted messengers, conflict mediation, and relationship-based outreach. However, many CVI organizations operate with limited analytical capacity. This tool aims to:

  • Identify emerging violence hotspots before incidents escalate
  • Enable strategic resource allocation across neighborhoods
  • Provide objective risk assessments for grant applications and reporting
  • Support coordination between CVI programs and public health partners

This is a tool for prevention, not punishment. It explicitly rejects use for predictive policing, surveillance, or any punitive applications.

Academic Foundation

This project builds on academic research conducted for the DSC 680 capstone course at Bellevue University (December 2025). The original paper, "Predictive Modeling for Community Violence Intervention Decision Support," established the theoretical framework and ethical principles that guide this tool.

Key contributions beyond the academic work include:

  • Transformation from theoretical framework to functional software
  • Implementation of multi-city adaptability
  • Creation of user-friendly web interface for non-technical users
  • Validation with real data collection from live public APIs
  • Development of comprehensive replication toolkit
